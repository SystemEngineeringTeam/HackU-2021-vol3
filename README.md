##  Installation

**開発をするには、前提条件としてNode.jsとGoLang、Docker/DockerComposeの導入が必須となります。**


初めに、GitHubからクローンします。

```
$ git clone git@github.com:SystemEngineeringTeam/HackU-2021-vol3.git

$ cd HackU-2021-vol3
```

次に、バックエンドの構築をします。

```
$ make all
$ cd backend
$ go run main.go
```

更に、フロントエンドの構築をします。

```
$ cd ..\front\
$ yarn
$ yarn dev
```

これで、開発を行うことが出来ます。
